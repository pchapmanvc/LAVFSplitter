                        VCExtras
                        ========

Introduction
------------

The VCExtras component allows playback of GXF, LXF and Matroska files in
Windows DirectShow programs.

It is a modified (and also simplified) version of the "LAV Filters" project
by Hendrik Leppkes (http://1f0.de) which in turn uses FFMPEG libraries for
decoding.

Source code for VCExtras is available at:

http://github.com/pchapmanvc/LAVFSplitter

Please direct enquiries to support@vidcheck.com
